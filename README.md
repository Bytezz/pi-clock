# pi-clock
Set alarms, create timers or stopwatches right from your terminal.


## Usage
### Setting an alarm
```bash
 piclock -a
```

### Creating a timer
```bash
piclock -t
```

### Creating a new stopwatch
```bash
piclock -s
```

### Interactive mode
```bash
 piclock -i
```
